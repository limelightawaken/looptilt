# LoopTilt Frontend

Next.js app for the LoopTilt creator workspace. See the [root README](../README.md) for full setup, architecture, and demo instructions.

```bash
npm install
echo 'NEXT_PUBLIC_BACKEND_URL=http://localhost:3001' > .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
